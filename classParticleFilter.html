ParticleFilter.cpp

A constructor for ParticleFilter class.

This is constructor for ParticleFilter Class to construct a tracker. frame0 is image matrix from getFrame(..) function. detidx is index of detection frame is current number of frame detectionBB is bounding box for a detection use_hist = 1, calculate color appearance model without dividing it into 3 regions use_hist = 2, calculate color appearance model by dividing it into 3 regions use_hist = 3, calculate color appearance model by dividing it into 3 regions and overlap each other imgcols is width of frame image imgrows is height of frame image
